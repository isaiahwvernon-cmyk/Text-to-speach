# TOA IP-A1 Speaker Control

A simple web interface for controlling TOA IP-A1 classroom speakers.

## Quick Start (Windows)

1. Install Node.js from https://nodejs.org (click the big green button, run the installer)
2. Download and unzip this project folder
3. Double-click **start.bat**
4. Open your browser to **http://localhost:5000**

## Quick Start (Mac / Linux)

1. Install Node.js from https://nodejs.org
2. Download and unzip this project folder
3. Open a terminal in the project folder
4. Run: `bash start.sh`
5. Open your browser to **http://localhost:5000**

## How to Use

1. Enter your speaker's IP address, username, and password
2. Use the volume slider, +/- buttons, or presets (Low / Normal / Loud)
3. Click Mute to silence the speaker

Your connection details are saved in the browser so you don't need to re-enter them.

## Requirements

- Node.js 18 or newer (https://nodejs.org)
- The computer running this must be on the same network as the speakers
